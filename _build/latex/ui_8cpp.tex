Description \+:

This node acts as the interface between the user and the robot. It communicates with the controller node to control the robot in 3 ways\+: autonomous driving, manual driving or assisted driving. To ensure the program does not block while waiting on certain types of input, this node works with the rt1a3\+\_\+inputasync node.
